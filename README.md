# screenscraper-tools
outils en ligne de commande pour gérer sa collection de jeux

# Bibliothèques requises
sudo apt install build-essential libtinyxml2-dev

# Identifiants développeur screenscraper
Pour pouvoir compiler ce programme, vous aurez besoin d'identifiants développeurs sur le site screenscraper.
Si vous n'en avez pas, vous pouvez toujours utiliser les programmes prédéfinis fournis en binaire.

# Configuration avant la compilation
Si vous voulez modifier et compiler ce programme, vous aurez besoin d'identifiants développeur sur le site screenscraper.

Dans votre fichier ~/.bashrc, ajoutez vos informations d'identifications à la fin du fichier 
export SSDEVID=XXXXX
export SSDEVPASSWD=XXXXX
export SSDEVPASSWDDEBUG=XXXXX
export SSID=XXXXX
export SSPASSWD=XXXXX
Enregistrez et relancez votre terminal
